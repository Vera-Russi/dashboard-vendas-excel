# Dashboard de Vendas - Projeto Xbox

## 📋 Descrição
Este projeto transforma dados brutos de vendas em um dashboard interativo no Excel, permitindo analisar faturamento, produtos mais vendidos e performance de vendedores.

## 📊 Insights Gerados
- Evolução mensal de vendas.
- Ranking de categorias e produtos.
- Filtros interativos por período.

## 🛠️ Tecnologias
- Microsoft Excel (Tabelas Dinâmicas, Gráficos Dinâmicos e Slicers).
